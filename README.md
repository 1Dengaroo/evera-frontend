# Evera Frontend

### Set up

Make sure you are using the right Node version defined on .nvmrc and .tool-versions

Create your .env file. Right now, it's only the API url.

```
cp .env.example .env
```

```
npm install
```

### Run

```
npm run dev
```

### Lint

```
npm run lint
```

or

```
npm run lint:fix
```

### Jest

We will implement this in the near future
